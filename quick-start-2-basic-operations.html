INSERT and UPDATE

As we've already seen, inserting a row is as simple as calling Ebean.save() with the object as an argument.

{% highlight java %} ... import com.avaje.ebean.Ebean; import com.avaje.ebean.SqlRow; import junit.framework.Assert; ... public class AppTest extends TestCase { ... public void testEntityCreation() { Customer c = new Customer(); c.setFirstName("Alice"); c.setLastName("Smith"); Ebean.save(c); ... } } {% endhighlight %}

The resulting SQL looks like this:

{% highlight sql %} insert into customer (id, first_name, last_name) values (?,?,?); --bind(1,Alice,Smith,) {% endhighlight %}
