An entity class is a proxy for a database table. Simply create a POJO and annotate with @Entity. A primary key column can be annotated by @Id.

{% highlight java %} import javax.persistence.Entity; import javax.persistence.Id; @Entity public class Customer { @Id private Long id; private String firstName; private String lastName; // Getters and setters... } {% endhighlight %}
