Start with an empty maven project.

{% highlight bash %} > mvn archetype:generate -DgroupId=org.avaje.ebeanorm.quickstart -DartifactId=ebean-quickstart -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false {% endhighlight %}

Now that your project is created, let's build it to make sure it's ready. {% highlight bash %} > cd ebean-quickstart > mvn install ... [INFO] ------------------------------------------------------------------------ [INFO] BUILD SUCCESS [INFO] ------------------------------------------------------------------------ {% endhighlight %}

Execute project

The resulting JAR can be executed using the "java" command. However, to make this easier, let's include the Exec Maven Plugin.

{% highlight xml %} org.codehaus.mojo exec-maven-plugin 1.2.1 org.avaje.ebeanorm.quickstart.App {% endhighlight %}

The full pom.xml should now look like this:

{% highlight xml %} 4.0.0 org.avaje.ebeanorm.quickstart ebean-quickstart jar 1.0-SNAPSHOT ebean-quickstart http://maven.apache.org junit junit 3.8.1 test org.codehaus.mojo exec-maven-plugin 1.2.1 org.avaje.ebeanorm.quickstart.App {% endhighlight %}

Let's execute "Hello World!".

{% highlight bash %} > mvn install > mvn exec:java ... Hello World! [INFO] ------------------------------------------------------------------------ [INFO] BUILD SUCCESS [INFO] ------------------------------------------------------------------------ {% endhighlight %}
